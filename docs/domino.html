Updating Domino Templates with Template 3

The Domino Developers in RTP have made a Domino Template for template 3.2.1 and guidance available.

There is a downloadable ZIP file of the template and an example database. The uses subforms with Javascript code for both the Northern Light search and a Domino database search. Please note that the files have been updated, but the above guidance has not been fully updated to match the new template, so there may be a couple things that don't quite match up anymore. The example database should help clarify.
