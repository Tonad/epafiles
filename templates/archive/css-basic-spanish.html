  1. Links to pages in English should either
    • use English link text. Note how the footer at the bottom of this page includes both Spanish and English link text. The English link text indicates links to pages in English. The bottom of the Superfund en Español home page provides an example of extensive use of English link text.

      OR
    • be followed by "(inglés)". Parts 5 and 11 in the sidebar at left provide examples.

      OR
    • be accompanied by the graphic located at http://www.epa.gov/espanol/layout-files/boton_eningles.gif. The alt text for this graphic should be "Información en inglés".
  2. External links should be accompanied by the graphic located at http://www.epa.gov/epafiles/images/exitepa-es.gif. This graphic should link to http://www.epa.gov/espanol/exit.htm. The alt text for this graphic should be "Salida y denegación".
  3. Links to PDF files should be accompanied by the graphic located at http://www.epa.gov/espanol/layout-files/pdf-icon-span.gif. This graphic should link to http://www.epa.gov/espanol/pdf.htm. In addition, every link to a PDF file should provide the file size and number of pages.

    Link text ( xx KB, yy pp.)

    Link text ( xx KB, yy pp.)
