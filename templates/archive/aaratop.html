AA/RA Template Pages

  1. This template is an enhanced AA/RA/Topic Homepage template. This template applies only to the home page of an AA/RA office or a Major Topic (e.g., Mercury). Contact your Content Coordinator to determine if your site is a "major topic".
  2. This template is fixed width (960px) and centered. The white space for your content is 800px wide.
  3. The banner is part of this template.
    • Modify the banner code where indicated and replace the banner image with your own by changing the path.
      • Edit this line: #header { background: url('index.jpg'); } by replacing index.jpg with the link to your own header image
    • Recommended dimensions for the image are 960 x 135.
    • The images should be GIF or JPEG (Standard: Graphic Formats)
    • Some panoramic images are available (intranet).
  4. There is no page title on home pages.
  5. Quick Finder boxes must follow the look of the EPA Web standard, but it doesn't matter how you create them.
