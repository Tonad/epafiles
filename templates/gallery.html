How To

Note the following:

  • The entire unordered list has a class of "gallery"
  • The first list item, in a row, has a class of "first"
  • The last list item, in a row, has a class of "last"
  • You can repeate for as many rows as you want
  • Each image is 128px wide, so you can fit up to five images per row.

If you want to use a popup to show the full-sized images, check out the jQuery Colorbox plugin.

Code

<ul class="gallery">
 <li class="first">
  <img src="http://www.epa.gov/epafiles/images/playset.jpg" width="128" alt=""/>
  <p>Studies have shown that pharmaceuticals are present in our nation's waterbodies.</p>
 </li>
