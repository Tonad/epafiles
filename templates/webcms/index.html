    • MS HP, four columns, equal widths
    • MS HP, three columns, differing width

    • MS HP, one column
  • Spanish Microsite Home Page (Micro-site-es template)
    • Espanõl HP
  • Resource Directory Pages
    • RD Home page (rdtropo template)
    • RD Basic (inner) (rdbasic template)
    • RD Basic (inner), two columns (rdColumns template)
  • All other inner pages (MS and channel)
    • Basic, one column (110-basic template)
    • Basic, two columns (210-basic two column template)
    • Basic, wide (120-basic-sidebarless template)
    • Basic, sidebarless (but not wide) (is not a template. Add a local style to areanav to create.)
    • Basic, Espanõl (110_es_basic template)

Microsites use:

  • microsite template for homepage
  • 110-basic for interior pages
  • 2 column and sidebarless as needed
  • note: don't use 2 column just to align boxes. It can be done in 110-basic.

Resource Directory use:

  • rdtropo template for homepage
  • rdbasic for interior pages
  • rdcolumns as needed
  • note: don't use rdcolumns just to align boxes. It can be done in rdbasic.

Channels use:

  • rdtropo template for homepages/landing pages
  • Laws and Regs use 110-basic for interior pages
  • Learn the issues and Science and Technology use rdbasic pages
  • Note: You will need to add a local style to the areanav file to hide the sidebar
  • Note: You can't have sidebars on some and not on others. Once style is there, no page will not have a sidebar
