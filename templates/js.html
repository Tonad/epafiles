Note: EPA no longer updates this information, but it may be useful as a reference or resource.

EPA builds all web content in the Drupal WebCMS as of January 2013. All new microsites and resource directories will be created using Drupal WebCMS. There is still content on EPA's legacy servers and this content will be maintained there until it is transformed and moved into the Drupal WebCMS. This information on styles and look and feel only applies to the existing content using Template 4 or older versions of EPA's template. Information for building new content is found in the EPA Web Guide.

"New!" icon

EPA's new icon is included as part of the core JS file (epa-core-v4.js). The code is identical for Template 4 and 3.2.1.

JavaScript frameworks, like jQuery, Prototype and Moo Tools, are designed to help us. These web frameworks sit on top of JavaScript and make some common tasks easier. When you write in JQuery, as you will in this tutorial, you're still writing JavaScript, but what used to be 50 lines may now be one or two. jQuery's slogan is "write less, do more," and it lives up to that promise.

The popularity of JavaScript frameworks, especially jQuery, means we potentially have multiple copies of the same JS files scattered throughout our web site. We are providing a standard suite of JavaScript files for everyone at the Agency to use. Linking to the EPA master JS files will improve our site performance, as users will cache a copy of the JS files. We're also providing a set of jQuery plugins.

All Template 4 pages link to the master jQuery file. If you want to add additional plugin functionality, you must also link to the plugin script. You can, of course, link to your own scripts if you don't find what you need here.

Please note, all JS should be linked at the bottom of the HTML, just before the closing </body> tag. Placing scripts here will speed up epa.gov for our readers. If your scripts are linked at the top, it would be good practice to adjust your templates.

Getting Started with jQuery

We've adapted a jQuery tutorial from webmonkey. Check it out!

In this tutorial, you find a new way to write JavaScript: the easy, JQuery way. You'll retrieve multiple objects in a single line, add CSS classes and make things magically disappear. At the end, you'll be able to create an expandable Frequently Asked Questions template, where users can click a question and the answer will appear below.

You don't have to be a pro to master jQuery. Even if you're new to JavaScript, you'll want to read on and see what JQuery can do for you.

JavaScript How-to

To create a slideshow, place the following code just before the </body> closing tag of your web page (or in the JavaScript tab of Web Publisher):

<script src="http://www.epa.gov/epafiles/js/third-party/jquery.cycle.lite.min.js"></script>

<script>

  $(function(){

  // Cycle a banner, replace "ID" with the name of the ID you want to cycle

    $('#ID').cycle();

  });

</script>

... where ID is the ID of the element you're "cycling".

The above code links to the "lite" version of the Cycle plugin. If you want more transition options, link to the full script instead:

<script src="http://www.epa.gov/epafiles/js/third-party/jquery.cycle.all.min.js"></script>

More Cycle Documentation

The above examples are basic, see the cycle plugin demo and documentation for more options. EPA's home page has a nice example of a cycling banner (starting and stopping on click, showing the number of slides, different timing).

</script>

Tablesorting How-to

To sort tables, place the following code just before the </body> closing tag of your web page (or in the JavaScript tab of Web Publisher):

<script src="http://www.epa.gov/epafiles/js/third-party/jquery.tablesorter.min.js"></script>

<script>

  $(function(){

  // Sort this table

    $("#ID").tablesorter();

  });

</script>
where ID is the ID of the table you're "sorting".

To make the zebra stripe sort with the content, do the following:

$("#table1").tablesorter({widgets: ['zebra']});
(Note: you do not need the zebra class for your table if you use this widget.)

This example is very basic, see the tablesorter plugin demo and documentation for more.

Script How-to

To use these tabs, place the following code just before the </body> closing tag of your web page (or in the JavaScript tab of Web Publisher):

<script src="http://www.epa.gov/epafiles/js/third-party/jquery.tools.min.js"></script>
<script>
 $(function(){
  // Tabs
  $("ul.tabs").addClass("tabs-js").tabs( "div.pane > div" );
 });
</script>

... where tabs-js is the class name of the element you're making into tabs (required) and pane is the name of the element you're applying the tabs to. Inside the "pane" element, each of the child elements will become a "tabbed" section: only one of the child elements will be visible, until you click on the next tab.

HTML How-to

The HTML is straightforward:

<ul class="tabs">

  <li><a href="#tabs-1">Nunc tincidunt</a></li>

  <li><a href="#tabs-2">Proin dolor</a></li>

  <li><a href="#tabs-3">Aenean lacinia</a></li>

</ul>

<div class="pane">

  <div id="tabs-1">

    <p>Proin elit arcu, rutrum commodo, vehicula tempus....</p>

  </div>

  <div id="tabs-2">

    <p>Morbi tincidunt, dui sit amet facilisis feugiat....</p>

  </div>

  <div id="tabs-3">

    <p>Mauris eleifend est et turpis....</p>

  </div>

</div>

Script How-to

To create an accordion, place the following code just before the </body> closing tag of your web page (or in the JavaScript tab of Web Publisher):

<script src="http://www.epa.gov/epafiles/js/third-party/jquery.tools.min.js"></script>

<script>

  $(function(){

    //  Accordion Panels

    $(".accordion div").show();

    setTimeout ("$('.accordion div').slideToggle('slow');", 1000);

    $(".accordion h3").click(function(){

      $(this).next(".pane").slideToggle("slow").siblings(".pane:visible").slideUp("slow");

      $(this).toggleClass("current");

      $(this).siblings("h3").removeClass("current");

    });

  });

</script>

... where accordion is the class name of the element you're making into an accordion (required) and pane is the class of the elements you're hiding (can be anything).

HTML How-to

The HTML is simple. Use h3 to create the "clickable" panel headers:

<div class="accordion">

  <h3>Section 1</h3>

  <div class="pane">

    <p>Mauris mauris ante, blandit et, ultrices a....</p>

  </div>

  <h3>Section 2</h3>

  <div class="pane">

    <p>Sed non urna. Donec et ante. Phasellus eu ligula....</p>

  </div>

  <h3>Section 3</h3>

  <div class="pane">

    <ul>

      <li>List item one</li>

      <li>List item two</li>

      <li>List item three</li>

    </ul>

  </div>

</div>

Form Validation

To validate forms, place the following code just before the </body> closing tag of your web page (or in the JavaScript tab of Web Publisher):

<script src="http://www.epa.gov/epafiles/js/third-party/jquery.tools.1.2.min.js"></script>

<script>

  $(function(){

  // Validate this form

    $("#ID").validator();

  });

</script>
where ID is the ID of the form you're valdiating. See an example, using common EPA Contact Us form fields.

Using jQuery with Template 3

You can use these javascript files with Template 3. You'll need to link to the master jQuery file (already linked in Template 4).

<script src="http://www.epa.gov/epafiles/js/third-party/jquery.js"></script>

This script link must go above any links to jQuery plugins.

Note that it's (as usual) not this simple: the CSS file for Template 4 contains a lot more styles, especially for things like tabs and forms. Those styles are not directly available for Template 3 unless you copy them over (into a local stylesheet).

  $(function(){

    // Sort this table

     $("#ID").tablesorter();

     $('#differentID').cycle();
