Code Library

Note: EPA no longer updates this information, but it may be useful as a reference or resource.

EPA builds all web content in the Drupal WebCMS as of January 2013. All new microsites and resource directories will be created using Drupal WebCMS. There is still content on EPA's legacy servers and this content will be maintained there until it is transformed and moved into the Drupal WebCMS. This information on styles and look and feel only applies to the existing content using Template 4 or older versions of EPA's template. Information for building new content is found in the EPA Web Guide.

Please Note: We are moving away from Dreamweaver.

Throughout epa.gov, we reuse a lot of code. We offer some here in an effort to standardize. If you have other common Dreamweaver Library items you would like to see here, please let us know.

To save these files, right-click on a link below and select "Save Target as.." (Internet Explorer) or "Save Link as..." (Firefox) from the pop-up menu. Save as type: All files. Then change the extension to .lbi. Note: Save Dreamweaver library items in the Library directory of your Dreamweaver site.

These libraries and code snippets work equally well in Templates 3 and 4.
