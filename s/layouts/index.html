Layouts

There are multiple ways of laying out your pages with CSS. Here are some simple ones, including . Use the one that suits you best and put it in a local style sheet. Experiment!

  • Default layouts. These layouts do not require local styles and will be used in EPA's forthcoming Web CMS:
    • Two column layout
    • Three column layout
  • Alternative two column layouts. These layouts require local styles:
    • Two columns: main column with right margin and first in source; sidebar absolutely positioned
    • Two columns: main column floated left; sidebar with large left-margin
    • Two columns: main column floated left; sidebar, floated left
  • Alternative three column layouts. These layouts require local styles:
    • Three columns: main column with right and left margins; sidebars absolutely positioned
    • Three columns: main column with right and left margins and last in source; left sidebar floated left; right sidebar floated right
  • Online resources:
    • Yahoo! Developer Network: Grids CSS
    • One clean HTML markup, many layouts...
