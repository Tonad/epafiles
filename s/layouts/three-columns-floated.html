Layouts

Floated Three-Column Layout

To recreate this layout, insert the following into your local style sheet (or into the <head> of this document):

<!-- Embedded styles for main and sidebar columns -->
<style media="screen,print" type="text/css">
	#sidebar1 { float: right; width: 200px; }
	#sidebar2 { float: left; width: 200px; }
	#main { margin: 0 215px; }
</style>

You, of course, can adjust all values to your liking.

Use the following structure in your content:

<div id="sidebar1">
	Right sidebar content here
</div><!-- END sidebar1 -->
<div id="sidebar2">
	Left sidebar content here
</div><!-- END sidebar2 -->
<div id="main">
	Main content here.
</div> <!-- END main -->
